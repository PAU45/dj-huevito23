# RENDER CONFIGURATION REFERENCE
# Copia esta configuración en tu dashboard de Render

## Web Service Settings

### Build Command
bash -lc 'npm ci && cd frontend && npm ci && npm run build && cd ../backend && npm ci'

### Start Command
node backend/index.js

### Environment Variables
# Agrégalas en Settings → Environment

DISCORD_TOKEN=PASTE_YOUR_BOT_TOKEN_HERE
ADMIN_API_KEY=PASTE_YOUR_API_KEY_HERE
PORT=3001
NODE_ENV=production

## Optional
# Si quieres restaurar cookies automáticamente en cada deploy:
# COOKIE_B64=PASTE_BASE64_ENCODED_COOKIES_HERE
# (Obtén esto de tu panel de cookies, en la sección "Export")

---

## Health Check Settings (Opcional)
Si Render pide configurar health checks:
- Type: HTTP
- Path: /
- Protocol: HTTP
- Check Interval: 10 minutes

---

## Deploy Hook (Si quieres auto-deploy desde GitHub)

1. En Render Dashboard → Settings → Build & Deploy
2. Copia el URL del Deploy Hook
3. En GitHub → Settings → Webhooks
4. Agregar webhook:
   - Payload URL: [URL del deploy hook de Render]
   - Content type: application/json
   - Events: Push events
   - Active: ✓

---

## Monitoreo de Logs

En Logs, busca estas líneas que indican todo funciona:

✅ "Servidor backend escuchando en puerto 3001"
✅ "Iniciando bot.js como proceso hijo..."
✅ "Bot process iniciado con PID [NUMBER]"
✅ "Bot conectado como dj huevito#4580"

---

## Restart Service

Si necesitas reiniciar el servicio:
1. En Render Dashboard → tu servicio
2. Botón "..." → Manual Deploy → Redeploy

---

## Región & Performance

- Región recomendada: La más cercana a ti
- Memory: 512 MB mínimo (por defecto está bien)
- CPU: El que viene por defecto
